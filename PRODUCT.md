# Product

## Register

product

## Users
Freelancers who want to quickly generate professional, platform-specific proposals that win jobs.

## Product Purpose
BidCopy is a functional tool that generates high-quality, tailored proposals for various freelance platforms (Upwork, Freelancer, Contra, etc.) to save freelancers time and improve their win rates.

## Brand Personality
Professional, fast, efficient, and reliable.

## Anti-references
Generic SaaS templates (e.g., typical cream/sand backgrounds with empty "supercharge" copy). It should not feel like an enterprise software tool, but rather a direct, no-nonsense utility for independent workers.

## Design Principles
- **Clarity over cleverness**: The UI should get straight to the point. No fluff.
- **Speed is a feature**: The generation and editing process should feel instant and frictionless.
- **Expert confidence**: The tool should instill trust through clean, structured typography and layout.
- **Platform-native feel**: Features like the Humanise Checklist should feel deeply integrated and context-aware.

## Accessibility & Inclusion
- Standard WCAG AA compliance for contrast and readability.
- Support for keyboard navigation in the editor and interactive checklists.
